### Tecnologia usada
---
* <a href="https://nextjs.org/">Next.js</a>

### Baixar e usar este projeto
---
```bash
    # Baixando o projeto
    $ git clone https://github.com/patrickrios/pomodoro-nextjs
```
```bash
    # Mudando para o diretório do projeto
    $ cd pomodoro-nextjs
```
```bash
    # Instalando as dependências
    $ yarn install
```
```bash
    # Executando o projeto
    $ yarn dev
```
